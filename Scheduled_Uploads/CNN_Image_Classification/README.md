# CNN Image Classification

Build a CNN to classify Cats vs Dogs images.

## Files
- `image_classification_cnn.ipynb` - Jupyter notebook
- `train_simple.py` - Training script  
- `predict.py` - Prediction script
- `SUMMARY_REPORT.md` - Learning outcomes

## Setup
```bash
pip install -r requirements.txt
```

## Train
```bash
python train_simple.py
```

## Predict
```bash
python predict.py image.jpg
```

## Results
- Accuracy: 80-90%
- Time: 15-30 minutes
