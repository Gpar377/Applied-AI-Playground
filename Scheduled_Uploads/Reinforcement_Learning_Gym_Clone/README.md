# Reinforcement Learning Gym Clone

This project implements a self-learning agent for grid navigation using reinforcement learning.

## Features

- Custom grid world environment simulation
- Agent trained with Q-Learning or PPO algorithms
- Visual dashboard to monitor episodes and agent performance
- Optional TensorBoard logging for training metrics

## Skills Demonstrated

- Reinforcement Learning (RL)
- Agent design and simulation
- Visualization and monitoring

## Setup

1. Install dependencies including stable-baselines3.
2. Run environment simulation and train agent.
3. Visualize training progress and results.
