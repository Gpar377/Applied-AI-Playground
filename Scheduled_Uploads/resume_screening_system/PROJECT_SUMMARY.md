# PROJECT SUMMARY REPORT

## AI-Powered Resume Screening & Job Matching System

**Project Type:** Intermediate AI/ML Application  
**Domain:** HR Tech, Natural Language Processing, Recruitment Automation

### Overview
This system intelligently analyzes resumes and matches them with job descriptions using advanced NLP techniques, going beyond simple keyword matching to understand semantic similarity and skill relevance. It provides explainable AI insights showing why a resume matched and what skills are missing.

### Core Components Implemented

1. **Resume Parser** - Extracts text from PDF/DOCX/TXT formats
2. **NLP Preprocessing Pipeline** - Tokenization, lemmatization, stopword removal
3. **Skill Extraction Engine** - Identifies technical and role-based skills
4. **Matching Algorithm** - TF-IDF vectorization + Cosine Similarity (60%) + Skill Match (40%)
5. **Ranking System** - Sorts candidates by relevance score
6. **Explainable AI** - Shows matched skills, missing skills, and score breakdown
7. **REST API** - FastAPI with 3 endpoints for production integration
8. **Web Interface** - Streamlit UI for user-friendly interaction

### Technical Stack
Python, NLTK, scikit-learn, FastAPI, Streamlit, PyPDF2, TF-IDF, Cosine Similarity

### Key Achievements
- Semantic understanding beyond keyword matching
- Production-ready API architecture
- Real-time resume ranking for multiple candidates
- Explainable scoring system for transparency
- Industry-standard NLP pipeline design

### Real-World Impact
This system replicates functionality used by major ATS platforms and HR tech companies, demonstrating practical AI application in talent acquisition and recruitment automation.
